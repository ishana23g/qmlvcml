Welcome to QMLvCML's documentation!
=========================================================

This is a python Package for the comparison of a quantum machine learning model with a classical machine learning model. 
The quantum algorithm is implemented using PennyLane Variational Classifier and the classical algorithm is implemented using SVM from scikit-learn.
